Hello there, I'm playing with GiHub features to create a user page!

Note a "user" page is not the same as "project" page.

  • Click the New repository icon from your account homepage
  • Name your repository .github.io, replacing with your GitHub username
  • Create an index.html page by clicking "Create new file" icon
  • Name the new file "index.html"
  • Enter the contents of the file and commit message
  • Click "Commit new file"
